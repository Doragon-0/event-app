# event-app
Projet TypeScript

Présentation de l'application. Ceci est une application de gestion d'événements où un utilisateur peut s'enregistrer à un événement créé par un adlinistrateur de l'application.

Ce projet est réalisé en HTML/CSS/TypeScript


Installation & lancement 
1) Ouvir l'invite de commande à la racine du projet

2) Compiler le TypeScript 
	tsc --watch 
3) Ouvrir le projet 
	npm start


Mode d’utilisation de l’application 

	Créer un événement
		
		Pour créer un événement il faut se connecter en tant qu'administrateur en renseignant les champs 	suivant:
			adresse: Admin
			mot de passse: Admin;

		Ensuite, sur l'interface qui s'affiche, renseigner les informations de l'événement et valider
	
	Comment filtrer ou rechercher ? 
		
		le filtrage se fait par catégorie. Il suffit de saisir la catégorie de l'événement recherché

	Comment s’inscrire ? 
		Pour s'inscrire, il faut au préalable avoir été enregistré en tant qu'utilisateur. Donc se rendre sur cette page et créer un utilisateur.
		Se rendre ensuite sur la page d'inscription à un événement, renseigner son adresse mail et l'identifiant de l'événement auquel on veur s'enregistré	

	Que se passe-t-il si l'événement est plein ?

		Lorsqu'un événement est plein l'inscription à celui-ci devient impossible
